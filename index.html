3、自定义排序
想降序排序，从大到小
static void sort(T[] a, Comparator< super T> c):

public interface Comparator {
//返回值是 int，我们根据返回值 >0、=0、<0 进行重新排序
int compare(T o1, T o2);
}

第一个参数：o1 > 第二个参数：o2 则执行的是升序
o1< o2 则执行降序

import java.util.*;

public class ArraysSortTest2 {
public static void main(String[] args) {

    Integer[] intArr = { 10, 20, 15, 22, 35 };

    // 自定义排序
    Arrays.sort(intArr, new Comparator<Integer>(){
        public int compare(Integer o1, Integer o2){
            // 判断 o2>o1，执行降序排序
            return o2-o1;
        }
    });

    System.out.println("Integer Array: "
                       + Arrays.toString(intArr));
}

}

1.3访问控制修饰符

（1）公开级别： public修饰，对外公开
（2）受保护级别： protected修饰，向子类及同一个包中的类公开
（3）默认级别： 没有访问修饰符，向同一个包中的类公开
（4）私有级别： private修饰，只有类本身可以访问，不对外公开

8.8 ArrayList
1、int 的 Java 对象就是 Integer；boolean 的 Java 对象就是 Boolean；double 的 Java 对象是 Double
2、有 Java 对象和没有对象的区别在于有对象的默认值是 null，比如 int 的默认值是 0，但是 Integer 的默认值是 null；boolean 的默认值是 false，但是 Boolean 的默认值是 null
3、由于 N 个文件的个数不确定，所以我们无法用数组来存储多个文件（因为数组的长度是事先确定的），这个时候我们就需要一个动态数组的方案来存储多个文件，这个动态数组在 Java 当中就是 java.util.ArrayList 对象

 **4、ArrayList语法**
 使用 ArrayList 就必须要先实例化，实例化的语法都是一样的，使用 new 关键字，并且需要指定对象类型
 // 这里的 Java 对象类型可以是任意的对象类型
// 比如 String、Integer、House 等

（这个对象是放在java.util包下的。所以要是用需要先进行import）

 **5、add方法**

 public class ArrayListTest1{
 public static void main(String[] args){

6、get/size方法
ArrayList具备数组的特点：（1）可以获取长度size（）； （2）可以根据索引获取具体的值 get(索引)，ArrayList 的索引是从 0 开始的
import java.util.ArrayList;
public class ArrayListTest1{
public static void main(String[] args){
// 创建一个 ArrayList 存储字符串集合
ArrayList strs = new ArrayList<>();

// 添加数据到 ArrayList 实例里
strs.add("张三");
strs.add("李四");

// 获取集合的长度
int size = strs.size();

// 使用 for 循环迭代每一条记录
for(int i=0;i<size;i++){
   // 根据索引获取值，值的类型是 String
   String str = strs.get(i);
   System.out.println(str);
}

}
