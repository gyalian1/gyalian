2、步数log（N），对数时间—O（log（N））
数组值获取（O（1））

3、猜数字游戏（O(logN))
O（log（N））比O（1）增长得更快一点，性能更差一点

时间复杂度为2N+2，用大O记法，线性时间为O（2N+2）

4、计算N的阶乘(O（N））
大O记法表示的是代码执行时间随数据规模增长的变化趋势，而在这种情况下，常量、系数部分并不左右增长趋势，所以可以忽略。
所以O（2 N+2）可以表示为O（N）

2.4性能优化小案例1-二分法查找

1、线性查找，依次遍历整个数组，判断给定的数字是否存在
public static int find(int[] array, int aim) {
for (int i = 0; i < array.length; i++) {
if (aim == array[i]) {
return i;
}
}
return -1;
}

2、二分查找法
定义两个变量left、right，分别表示数字可筛选区域左侧右侧的索引（下标）
public static int find(int[] array, int aim) {
// 初始化left = 最左侧, right = 最右侧
int left = 0;
int right = array.length - 1;

// 当left > right则表示遍历完成
while (left <= right) {
    // 获取中间的值
    int middle = (left + right) / 2;
    int middleValue = array[middle];
    if (middleValue == aim) {
        // 已经找到目标
        return middle;
    } else if (middleValue < aim) {
        // 目标在middle的右侧，重置left
        left = middle + 1;
    } else {
        // 目标在middle的左侧，重置right
        right = middle - 1;
    }
}
return -1;

}

public static ArrayList repeat(int[] array) {
ArrayList result = new ArrayList<>();
for(int i = 0; i < array.length; i++){
// 以此判断i位置元素和后面j位置元素是否相等
for(int j = i + 1; j < array.length; j++){

result.add(array[i]);
}
}
}
return result;
}

public static ArrayList repeat(int[] array) {
ArrayList result = new ArrayList<>();
int[] exists = new int[11];
for (int i = 0; i < array.length; i++) {
int value = array[i];
// 如果当前位置已经为1，则表示重复
if (exists[value] == 1) {
result.add(value);
} else {
// 否则将当前位置设置为1
exists[value] = 1;
}
}
return result;
}

新Java网络编程

1.1协议
1、网络协议是通信计算机双方必须共同遵从的一组约定。如怎么样建立连接、怎么样互相识别等。只有遵守这个约定，计算机之间才能相互通信交流。
2、目前网站只有HTTP/HTTPS协议

3、在浏览器的开头固定的字符：”http://“，意思是协议，在互联网上传输超文本
“https：//”区别是：有实力的网站提升了安全性、https://开头的网站要注意隐私信息

1.2 URL
1、地址栏输入的地址叫做URL，也就是网址
URL统一资源定位符
URL的格式规定有几部分组成以及各种符号的作用：

（1）协议类型与域名之间以 ://（固定写法）分隔。
（2）路径（英文常称为 path）以单斜杠 / 开头，中间每层的分隔符也是单斜杠 /
（3）参数：
路径与参数之间用 ? 分隔。看到问号 ? 就知道后面的内容就是参数了。
多个参数之间用 & 分隔
参数用“参数名=参数值”（key=value）的格式表示。

2、路径的两种情况：
gallery/topic/116390/?from=hot_topic_note&sort=new
相对路径：不是以斜杠 / 开头的路径，表示相对路径
默认路径：实际上，没有输入路径时，表示请求网站的默认页面，那么服务器就会返回一个默认页面给浏览器。至于具体默认页面是什么页面，是由服务器决定的，通常服务器默认的页面是首页。

//返回值是 int，我们根据返回值 >0、=0、<0 进行重新排序

第一个参数：o1 > 第二个参数：o2 则执行的是升序
o1< o2 则执行降序

import java.util.*;

        public int compare(Integer o1, Integer o2){
            // 判断 o2>o1，执行降序排序
            return o2-o1;
        }
    });

    System.out.println("Integer Array: "
