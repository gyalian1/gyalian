6.1 自增和自减
1、自减
自减的语法是：变量名称–，比如 number–;
–的两个减号等于number= number-1
实际上只要固定减数字的都可以采用自减的语法，但是又有一点的区别：
number = number -2 可以写成 number-=2
2、自增
自增的语法是：变量名称++，比如number++
++两个加号等于number= number+1
number = number+2 可以写成number+=2

6.3 使用for循环
1、循环的知识相对有一点点抽象，简单的来说就是有个计数器一直在递增，递增到指定的值为止，在数组中，我们可以通过这个计数器的值得到数组的对应下标的值
2、循环的语法就是 for 语句，for 语句中小括号的内容就是指定了计数器如何工作
3、循环语句：
for(int i = 0; i< tables.length;i++){
String name = tables[i];
System.out.println(name);
}

(int i =0 ; 这句声明了一个新的计数器，它是从0开始工作的)
（i< tables.length; 这句声明了计数器的条件，最大值不能超过 table.length）
(i++ 这句声明了计数器的工作机制，是递增，每次+1)
4、简化写法：
原始：
public static void main(String[] args) {
String[] tables = new String[3];
tables[0] = “张三”;

tables[2] = “王五”;

for(int i=0; i < tables.length; i++){
String name = tables[i];
System.out.println(i+”：”+name);
}
}
简化：
public static void main(String[] args) {
String[] tables = new String[]{“张三”,”李四”, “王五”};
for(int i = 0; i < tables.length; i++){
String name = tables[i];
System.out.println(i+”：”+name);
}
}

(new String[]{“张三”,”李四”, “王五”}; 这个语法就是初始化字符串数组数据 ，如果是数字数组：new int[]{39,40,89}; 浮点数组是 new double[]{39,40,89};)

7.3 字符串分割
1、字符串分割spilt
多数时候处理这样的数据，会先把这种文本数据变成数组，每一行作为一条记录，从一个文本变成数组数据
split(“分割字符串”)
这个分割字符串可以是字符串也可以是特殊的符号，比如说换行的符号是\n
split 方法执行的结果是返回一个字符串数组对象

public static void main(String[] args){
String text = “姓名|年龄|性别\n张三|20|男\n李四|18|男\n小花|18|女”;
// 使用 split 进行换行符的分割，得到一个新的数组对象
String[] data = text.split(“\n”);
// 因为第一行是标题不是数据，所以我们需要把长度-1
// (注意要使用小括号包围，因为要先计算长度再组合字符串)
System.out.println(“共有:”+(data.length-1)+” 条记录”);
}

GitHub笔记

1、绑定SSH:
（1）查看是否生成SSH ：cat ~/.ssh/id_rsa.pub
（如果有SSH则会显示，如果没有则会显示空白）
（2）生成SSH：ssh-keygen -t rsa -b 4096 -C “your_email@example.com“
（输入后按3次回车键，生成SSH，改成注册github的邮箱）
（3）生成SSH后去获取它的命令：cat ~/.ssh/id_rsa.pub
（4）复制代码到Setting里的SSH，添加到Key里去
（复制的文件以ssh-rsa为开头的）
电脑本地和Github服务器之间的SSH（安全协议外壳），可以保证本地电脑和Github之间的安全连接
2、全局变量设置
（1）查看当前目录下的文件：ls
（2）查看当前目录下的文件，以及隐藏文件： ls -a
（3）在当前目录下创建一个文件： touch + 文件名
（4）在当前目录下创建一个文件夹：mkdir + 文件夹名
（5）进入某个文件夹： cd +加文件夹目录
（6）删除： rm -rf
（7）使用git clone的命令将github上的仓库下载到本地电脑上：git clone 仓库地址
(仓库地址在仓库的code或者clone or download里面点击Use HTTPS的地址，以git开头的 )
（8）告诉git你的名字 ：git config –global user.name “名字”
（9）告诉git你的邮箱 ：git config –global user.email “邮箱”
[（8）（9）的配置只需进行一次 ]

3、Git的提交
（1）git的提交分三步：1、git add
2、git commit
3、git push
（2）git add 的命令为：git add -A
（3）git commit的命令为：git commit -m +”本次提交的修改的备注”
（知识修改，没有创建使用 git commit -am+ “本次提交的修改的备注”）
（4）退出命令行：ctrl+c
（5）git push 的三个命令：第一次提交到本分支：git push origin master
（origin是远程仓库的默认名称，master是主分支名称）
从第二次提交到本分支：git push
提交到其他分支：git push origin b
（6）git pull 是抓取远程仓库的内容
（多人协同开发时需要，每次提交前git pull一下，可以避免冲突）

4、绑定本地仓库到Github
（1）首先是在github创建一个同名的空仓库就不勾选方框
（2）绑定远程仓库：
1、首先cd进入文件夹
2、初始化为git仓库 git init
（是将一个文件夹初始化为一个空的git仓库）
3、绑定远程仓库
 查看当前git仓库绑定的远程仓库：git remote -v
（使用命令，显示空白才可以远程仓库绑定，否则会有冲突）
 用于绑定远程仓库 ：git remote add origin +仓库地址
绑定成功后再git remote -v查看一下
（1）绑定错误，删除绑定：git remote remove origin
 (2)最后提交：git add -A git commit -m + “备注” git push origin master
