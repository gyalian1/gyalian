List和数组的互换

1、出现List和数组共存的情况，需要进行数据的互传，java.util.Arrays是专门处理数组和集合的类
Arrays是一个工具类，它的方法都是静态方法。
2、最常用的转化方法：static List asList（T… a)
asList方法是把任意的相同对象实例转变为List
T… a：
T是泛型类型
泛型代表可以是任意的java对象，比如String、Course等，如果要使用的话，还需要在方法名前额外声明，比如

… 是可变参数的一种声明
操作这个参数变量可以当做数组来运用。这个方法的签名类似：static List asList(T[] a)

import java.util.Arrays;
import java.util.List;

public class ArraysMain {
public static void main(String[] args) {
// 数字数组
Integer[] intArr = { 10, 20, 15, 22, 35 };
// 转化数组为 list
List lists = Arrays.asList(intArr);
System.out.println(“集合的长度是: “ + lists.size());
}
}

Arrays.asList()的参数值必须是对象数组，例如：Integer[]

3、自定义排序
想降序排序，从大到小
static void sort(T[] a, Comparator< super T> c):

public interface Comparator {
//返回值是 int，我们根据返回值 >0、=0、<0 进行重新排序
int compare(T o1, T o2);
}

第一个参数：o1 > 第二个参数：o2 则执行的是升序
o1< o2 则执行降序

import java.util.*;

public class ArraysSortTest2 {
public static void main(String[] args) {

    Integer[] intArr = { 10, 20, 15, 22, 35 };

    // 自定义排序
    Arrays.sort(intArr, new Comparator<Integer>(){
        public int compare(Integer o1, Integer o2){
            // 判断 o2>o1，执行降序排序
            return o2-o1;
        }
    });

    System.out.println("Integer Array: "
                       + Arrays.toString(intArr));
}

}

return o1-o2;）

4、集合的排序
java.util.Collections，这个collection工具类也有一个sort方法来处理排序：
static void sort(List list, Comparator<? super T> c)

import java.util.*;

public class ListSortTest {
public static void main (String[] args) {
List ar = new ArrayList();
ar.add(new Student(111, “bbbb”, “london”));
ar.add(new Student(131, “aaaa”, “nyc”));
ar.add(new Student(121, “cccc”, “jaipur”));

System.out.println("原始集合");
for (int i=0; i<ar.size(); i++) {
  System.out.println(ar.get(i));
}

// 实现升序排序
Collections.sort(ar, new Comparator<Student>(){
  public int compare(Student a, Student b) {
    // 第一个参数的学号 > 第二个参数的学号
    return a.getRollNo() - b.getRollNo();
  }
});

System.out.println("\n排序后的集合");
for (int i=0; i<ar.size(); i++) {
  System.out.println(ar.get(i));
}

}
}

// 实现降序排序
Collections.sort(ar, new Comparator(){
public int compare(Student a, Student b) {
// 第二个参数的学号 > 第一个参数的学号
return b.getRollNo() - a.getRollNo();
}
});
