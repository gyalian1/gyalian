Exception

4.2Java异常类

1、常见的异常类
（1）IOException ：操作输入流和输出流时可能出现的异常
（2）ArithmeticException ：数学异常（如果整数除以0，就会出现这种异常 int a=12/0; //抛出ArithmeticException）
（3）NullPointerException ：空指针异常。（String d = null;
System.out.println(d.length()); // 抛出NullPointerException）
（4）ArrayIndexOutOfBoundsException ：下标越界异常，一般出现在数组里
int[] array = new int[4];
array[0]=1;
array[7]=1;// 抛出 ArrayIndexOutOfBoundsException
（5）ClassCastException ：当类型转化失败的时候就会出现该异常
（6）IllegalArgumentException ：非常参数异常，可以用来检查方法的参数是否合法
2、异常的作用域
public class ExceptionTest2 {

public static void main(String[] args) {
    int len = 0;
    try {
        len = getStrLength(null);
    } catch (NullPointerException e) {
    }

    System.out.println("程序执行到这"+len);
}

// 获取字符串的长度
public static int getStrLength(String str) {
    return str.length();
}

}

作用域的提升就是把代码块的变量在代码块之前先定义

3、多个异常捕获
想把某些字符串转换为数字就需要Integer.parseInt(str)方法
public class ExceptionTest3 {

public static void main(String[] args) {
    int num = getInt("20");
    System.out.println(num);
}

// 字符串转换为数字
public static int getInt(String str) {
    return Integer.parseInt(str);
}

}

想要捕获两种异常：
public class ExceptionTest4 {

public static void main(String[] args) {
    int num = 0;
    try {
      num = getInt("小王");
    } catch (NumberFormatException e) {
    } catch (NullPointerException e) {
    }

    System.out.println(num);
}

// 字符串转换为数字
public static int getInt(String str) {
    return Integer.parseInt(str);
}

}

由于这些Exceptions都是继承于Exception，所以可以直接用
public class ExceptionTest4 {

public static void main(String[] args) {
    int num = 0;
    try {
      num = getInt("小王");
    } catch (Exception e) {
    }
    System.out.println(num);
}

// 字符串转换为数字
public static int getInt(String str) {
    return Integer.parseInt(str);
}

}

4、例子
使用 split() 方法把字符串 content 转化为字符串数组
String content = “10,20,,50,小明”;
String[] a = content.split(“,”);
把字符串数组转化为 int 数组，如果字符串不是数字则用0替代
求和一下这个 int 数组，得到数组累加后的值，打印出求和后的值

public class ExceptionTest4 {

public static void main(String[] args) {
    String content = "10,20,,50,小明";

    String[] a = content.split(",");
    int[] nums = new int[a.length];

    int total=0;
    for(int i=0;i<a.length;i++){
        try {
              int num = Integer.parseInt(a[i]);
              nums[i] = num;
              System.out.println(nums[i]);
            } catch (Exception e) {
              nums[i]=0;
              System.out.println(nums[i]);
    }

    total +=nums[i];
    }
    System.out.println(total);
}

}
