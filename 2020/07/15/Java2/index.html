4.1 条件执行
1、编程的很多工作都是要进行各种条件的判断，
Java和其他编程语言一样支持条件语句，条件语句就是让我们执行条件判断的语法
2、关键字 if 后括号中的代码叫做：条件表达式，如果它为真，大括号中的语句就会执行；反之，则什么都不会发生
if(表达式){代码块}
3、条件表达式可以包含任意的比较运算符（有时候也叫做关系运算符）
x == y // x 等于 y
x != y // x 不等于 y
x > y // x 大于 y
x < y // x 小于 y
x >= y // x 大于等于 y
x <= y // x 小于等于 y
(=符号是赋值运算，==是判断是否相同的比较运算符)
3、在main中调用两次report方法，传出小U，98；小D，58
public class ConditionTest4 {
public static void main(String[] args){
report(“小U”,98);
report(“小D”,58);
}
public static void report(String name, double score){
if(score >= 60){
System.out.println(“恭喜”+ name+”通过了，” + “本次考试” + score + “分”);
}
if(score <=60){
System.out.println(name + “你还需要继续努力，本次考试” + score + “分”);
}
}
}

4.3 条件判断链
1、在某些时候我们会遇到条件判断多于两个的情况，并且需要根据条件的不同执行不同的分支，这个时候我们就需要新的技术：条件链
2、else if
条件链的执行是先从第一个条件判断开始，如果判断不成功继续执行下一个判断，一直到没有条件符合的情况。

}

5.2 布尔&&逻辑运算符号

2、除了数学表达式之外，Java 还支持三种逻辑运算符：and（与）、or（或者）、not（非），对应的符号分别是: && || !

（1）   if(x>0 && x <10){
             // 表示 x 大于0 并且 x小于10时才为真
           }

（2）  if(x == 60 || x == 80){
             // 表示 x 等于60 或者 x 等于80 都为真
           }

（3）  if(x!=0){
             // 表示x不等于0的时候
           }

6.1 自增和自减
1、自减
自减的语法是：变量名称–，比如 number–;
–的两个减号等于number= number-1
实际上只要固定减数字的都可以采用自减的语法，但是又有一点的区别：
number = number -2 可以写成 number-=2
2、自增
自增的语法是：变量名称++，比如number++
++两个加号等于number= number+1
number = number+2 可以写成number+=2

6.2数组
1、在编程语言里，除了前面说的字符串、数字、布尔这三个基础的数据类型外，其实还有一个基础数据类型，那就是数组，数组就是一组类型相同的数据组合在一起

数组的创建和普通变量的创建基本上差不多，唯一有差别的是数组类型后面多了一个[]（中括号）。比如，int[]表示“整型数组”类型，String[]表示“字符串数组”类型.

// 声明一个 int 数组的变量，数组大小为6

int[] numbers = new int[6];

创建好数组变量后，就需要存储每一个数据，那么我们可以借助[下标]来完成操作; 同样如果想获取数组的某一个数据，也是用下标

// 声明一个 int 数组的变量，数组大小为6

int[] numbers = new int[6];

numbers[0] = 1;

numbers[1] = 3;

numbers[2] = 5;

numbers[3] = 7;

numbers[4] = 9;

numbers[5] = 11;

int num = numbers[ X];

System.out.println(num);
2、长度
数组对象都有一个值length，可以得到该数组的长度，数组的坐标都是从0开始的，所以如果想去数组的最后一个数据，正确的下表是length-1
3、数组越界错误
数组长度是8，size就是8，但是只有1234567，所以size[8]是不存在的
4、初始化变量
public static void main(String[] args) {
int[] numbers = new int[8];
}
目的就是做了一个数组的初始化，这个初始化的作用就是开辟了该长度的存储空间

5、数组初始化完后，并没有存储实际的值，int 类型的数据默认值是 0，所以如果没有完成数组的赋值，那么 int 数组的每一个值都是 0；String 类型的数据默认值是 null，没有赋值，那么String数组的每一个值都是null
