Java笔记

1、Java 的概念

2、Java对于字符串的输出都是用英文的”引号
3、Java 文件名必须是英文字母（有数字也可以，尽量不用数字），并且不能有空格。一般来说 Java 文件名首字母是大写的，比如说ser.java、School.java、Book.java
4、Java 语言的进阶可以分为两层：Java 语言、JVM
5、源代码被编译成程序才可以被计算机执行
6、Java遵循两种代码注释：块注释和行注释
7、Java 是一个面向对象的语言,Java把一切内容（字符串、数字、布尔、日期、时间）都用对象来表示，所以在 Java的世界中变量内容就是对象
8、每个变量都需要一个类型来决定可以存储什么类型的数值。比如，int 类型可以存储整数，String 类型可以存储字符串
9、Java中常见的关键字有public、 class、 void、 int
10、在编程语言中，存储数据的行为叫做：赋值
11、实际上在所有的编程语言里，基本上都包含三种基础数据类型：
字符串、数字、布尔
12、字符串就是String，字符串用英文引号””
13、数字类型：小整数 int ； 大整数 long ；浮点数 double 15、小整数和大整数的区别在于存储的数据大小不一样，我们一般接触的数字都是小整数，所以用大部分时候 int 就可以了
16、long类型赋值的时候要在数字最后更上一个小写的l（是字母L)
17、布尔：boolean ，布尔的值只能是true 或者 false
18、在编程语言中加减乘除这些数学符号称为：
运算符
加+ 减- 乘* 除/ 整除运算符%
19、Java 执行的是整除除法，也就是说当两个操作数都是整数时，结果也会自动变成整数（直接去掉了小数点的值）
20、计算百分比的话System.out.println(100*now/xxx)
21、平均分的代码：int total = 0；
int u1 = x；total=total+u1；
int u2 = xx；total= total+u2；
int average = total/4；
System.out.println（average）；
22、Java 在数学上面的能力非常强大，有很多内置的数学函数，这些函数在 Java 当中称为方法（method），有一个数学函数是我们应该知道的，而且非常实用。那就是随机数
23、Math.random() 方法用于返回一个随机数，随机数范围为 0.0 =< Math.random < 1.0。
代码：double value = Math.random(); System.out.println(value);
或者是Math.random()+1
24、浮点数如何转化为整数，理论上数字类型都可以转化，只是会丢失掉精度（自动去掉小数点了）
Java中类型转换的语法：int nValue =（int）(value*10)
25、main 方法有特殊的含义（它是程序执行的入口,如果要执行代码就必须要有 main 方法）
main方法用于启动程序
26、紧跟着方法名的空小括号表明此方法不需要参数
27、自定义方法调用：代码编写的顺序，一定是先定义方法，再调用方法
28、自定义的方法都是在 class 类当中的，方法的顺序无所谓，放在 main 方法前面或者后面都可以，但是方法内是不能这样新建方法的
29、方法参数

30、方法参数和声明变量并无区别，只是这个变量是要定义在方法参数这个位置里，编程语言里把这个声明的变量叫做形参
31、方法参数
public static void main（String[] args）{
code(100000);
}
public static void code (int length){
int code =(int)((Math.random()+1)length);
System.out.println(“——“);
}

String text = “”;
code(text,100000);
}

int code =(int)((Math.random()+1)*length);
